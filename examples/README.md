# Relworx Python SDK Examples

This directory contains example scripts demonstrating how to use the Relworx Python SDK.

## Setup

1. Install the library:
   ```bash
   pip install relworx
   ```

2. Set your API key:
   ```bash
   export RELWORX_API_KEY="your-api-key-here"
   ```

## Examples

### example.py

Basic usage examples including:
- Requesting payments
- Sending money
- Checking transaction status
- Validating payment details
- Getting exchange rates
- Using context manager

Run it:
```bash
python example.py
```

## Getting Your API Key

1. Sign up at [Relworx Payments](https://payments.relworx.com/users/sign_up)
2. Create a business account
3. Generate your API key from the dashboard
